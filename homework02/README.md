# Homework 02

## About
The purpose of this homework was to get familiar with Git/GitHub and review Python with a few coding exercises.

### Exercises
**Exercise 1:** Determines whether a list of integers is even or odd.

**Exercise 2:** Determines the GC content of a DNA sequence using BioPython's Seq class. 

**Exercise 3:** Calculates the percentage of each base (A, T, G, C) in a DNA sequence.

**Exercise 4:** Calculates the mean expression, fold change, and significance for the control and treatment group of a list of genes.
