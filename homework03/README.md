# Homework 03

## About
The purpose of this homework was to get familiar converting between different file formats, including JSON, CSV, XML, and YAML.

### Exercises
**Exercise 1:** Reads in a JSON file and finds the total mass of the proteins, the large proteins, and the non-eukaryotic proteins. 

**Exercise 2:** Converts the protein data from the JSON file into CSV format.

**Exercise 3:** Converts the protein data from the JSON file into XML format.

**Exercise 4:** Converts the protein data from the JSON file into YAML format.
